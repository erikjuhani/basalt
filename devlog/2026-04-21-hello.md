I've been thinking about starting a devlog for some time now to provide updates
and insights on what is happening behind the scenes while I'm building
`Basalt`.

It might seem slow on the surface (well it is!), but I'm working on multiple
things, and in different domains around `Basalt`. For instance this devlog, and
the documentation site.

In this day and age it feels almost necessary to have a 'distinct'
documentation site for applications. So I decided already months ago that
`Basalt` should have a documentation site with `gifs` showcasing the different
features.

Building the `gifs` and `vhs` tapes took a long time. However, now that they
are set up the subsequent ones should be relatively easy to create or modify.

Additionally new software was created to transform Obsidian vault into zola
compatible markdown files. Perhaps more on that in another devlog.

Cheers!

— Erik
